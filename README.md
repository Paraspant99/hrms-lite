# HRMS Lite

A lightweight HRMS system to manage employees and daily attendance.

## Features
- Add/List/Delete employees
- Mark attendance (Present/Absent) per employee per day
- View attendance records per employee
- Attendance summary (presentDays / absentDays)

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MySQL
